# 🎮 Tic Tac Toe Game (Python)

## 📌 Project Description
This is a simple **Tic Tac Toe game** developed using **Python**.  
The game allows two players to play Tic Tac Toe in the terminal/console.

It demonstrates basic Python concepts such as:
- Functions
- Loops
- Conditional statements
- Lists
- User input handling

---

## 🚀 Features
- Two-player gameplay
- Console-based interface
- Win detection system
- Draw detection
- Easy and beginner-friendly logic

---

## 🛠️ Technologies Used
- Python 3
